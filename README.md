# 🎬 My Film App API

Aplikasi web sederhana berbasis Laravel yang menampilkan daftar film menggunakan Public API (Studio Ghibli API).  
Project ini dibuat untuk memenuhi tugas implementasi API pada aplikasi monolith.

---

## 🚀 Fitur Utama

- Menampilkan daftar film 🎥
- Halaman detail film
- Fitur pencarian film 🔍
- Tampilan UI menarik dengan animasi
- Menggunakan HTTP Client Laravel untuk konsumsi API

---

## 🛠️ Teknologi yang Digunakan

- Laravel 12
- PHP 8
- Blade Template
- CSS (Custom Styling)
- Public API (Studio Ghibli API)

---

## 🌐 API yang Digunakan

Menggunakan Public API tanpa API Key:

https://ghibliapi.vercel.app/films

---

## 📷 Tampilan Aplikasi

### Halaman Beranda
Menampilkan daftar film dengan poster dan animasi.

### Halaman Detail
Menampilkan informasi lengkap film:
- Judul
- Tahun
- Deskripsi
- Gambar
